# tools.simpleweb3.ch

## To Run 

### Pre-Requisites
> 1. Nodejs version 20+

### Installation

```bash
npm install
```

### Have a environment variable 
```bash
mv .env.example .env
# DONT FORGET TO UPDATE ENVIRONMENT VARIABLE 
# FETCH THE PROJECT ID FROM https://cloud.walletconnect.com/
```

### Start the website for development

```bash
npm run dev
```

### Start the website for production(Two Steps)

first build
```bash
npm run build 
```

then preview
```bash
npm run preview 
```
